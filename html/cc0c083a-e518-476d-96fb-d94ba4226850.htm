Install Hummingbird

Prerequisites

The Hummingbird project is built upon .NET Framework 4.5. You must have .NET Framework 4.5 or above already installed before running the software.

.NET Framework 4.5 is already installed with Windows 8 and Windows Server 2012, you don’t need to install on these operating systems, including all operating systems release after them.

If you are using Windows Vista SP2, Windows 7 SP1, Windows Server 2008 SP2 or Windows Server 2008 R2 SP1, please check and install the .NET Framework from this link.

Installation instructions

There are several ways to install Hummingbird according to needs.

When running Hummingbird on a Desktop machine for Test and Automation by using the GUI application, the best way is to install the ClickOnce version. It gives you a way to easy install (by one click) and follow the latest version. To install with ClickOnce, Please refer to ClickOnce Installation

If you want run Hummingbird on a Server Machine, For Testing, Automation and Command Line interface. You should use the Windows Installer or Zip version.

 Note

You can install multiple versions of Hummingbird from different media, But All hummingbird instances share the same copy of Profile, History and Settings.

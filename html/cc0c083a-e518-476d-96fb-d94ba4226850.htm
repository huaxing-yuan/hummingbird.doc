Prerequisites

The Hummingbird project is built upon .NET Framework 4.6. You must have .NET Framework 4.6 or above already installed before running the software.

The following operation systems have already .NET Framework 4.6 pre-installed.

  • Windows 10

  • Windows Server 2016

  • Or the newer version of Windows and Windows Server

When using following version of Windows, please make sure the .NET Framework 4.6 is installed. from this link.

  • Windows Vista SP2

  • Windows 7 SP1

  • Windows 8

  • Windows 8.1

  • Windows Server 2008 SP2

  • Windows Server 2008 R2 SP1

  • Windows Server 2012

  • Windows Server R2

Installation instructions

There are several ways to install Hummingbird according to needs.

When running Hummingbird on a Desktop machine for Test and Automation by using the GUI application, the best way is to install the ClickOnce version. It gives you a way to easy install (by one click) and follow the latest version. To install with ClickOnce, Please refer to ClickOnce Installation

If you want run Hummingbird on a Server Machine, For Testing, Automation and Command Line interface. You should use the Windows Installer or Zip version.

To install Hummingbird, please choose the following installation guide

 Caution

Download and install any version of Hummingbird, means you accept the License Agreement

  • ClickOnce Installer

  • Windows Installer

  • Zip Archive

 Note

You can install multiple versions of Hummingbird from different media, But All hummingbird instances share the same copy of Profile, History and Settings.

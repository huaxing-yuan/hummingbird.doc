HttpHeaderAttribute Class

Remarks
HttpHeader name is specified in Name Property of HttpHeaderAttribute. The value is the attached property's string value. For Enums, If the enum is attached with EnumMember attribute, Hummingbird Test Framework will take the value specified in EnumMember Attribute.

Extension Designer

Extension Designer

Extension Designer is used to import and configure Web Services specified in WSDL and Swagger

Import SOAP Services from WSDL

Extension Designer can import and compile extensions from a given WSDL, either on a network location or a local location. Currently, you can import Web Services both SOAP 1.1 and SOAP 1.2, encoded either in XML or MToM and different kinds of authentication process (None, Basic, Windows Integrated).

Import OpenAPI services from Swagger

Extension Designer will support also REST services described with Swagger. This function is under development.

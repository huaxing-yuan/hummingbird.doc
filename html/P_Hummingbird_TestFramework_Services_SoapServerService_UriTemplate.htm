Remarks
The virtualized services are hosted on where Hummingbird Test Framework is running. For example, you are running a service where: UriTemplate is /myservices/myservice on the Machine 192.168.1.110, you should configure your application to call this service to http://192.168.1.110:8080/myservices/myservice The port 8080 is the default port for virtualized SOAP Services, but you can change the port number in settings. For more information about Soap Vitualized service, please refer to SoapServer

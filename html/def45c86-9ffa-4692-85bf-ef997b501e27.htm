Profile management

It is possible to manage more than one test settings, that to pack them in different Profiles. A profile includes service parameters, test environments, custom variables credentials and service simulation behaviors.

Manage test profiles

Default profile

Default profile is that used by Manual Testing, to modify default profile, you need to go to Test Settings. Default profile can not be edited here and be deleted.

Other profiles

You can import profile from a file, or duplicate from existing profiles. Active profile can be used and embedded into an automation project

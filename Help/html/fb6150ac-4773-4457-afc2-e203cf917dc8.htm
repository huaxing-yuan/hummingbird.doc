Extension management

Extensions gives more possibility to test applications: it extends from existing protocols by importing services descriptions to test, it extends the test framework by implementing new test protocols.

Extensions are versioned and can be installed by yourself.

All installed extension are shown here, you can:

  • View information about an extension

  • Enable or disable an extension

  • Install new extensions. Hummingbird detects also installed extensions and makes update is necessary, and does its best to avoid application restarting.

  • Delete an extension. To delete an extension, you need to restart Hummingbird before the deletion is effective.

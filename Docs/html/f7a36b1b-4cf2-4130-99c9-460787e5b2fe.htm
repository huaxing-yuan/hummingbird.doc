Windows Installer

Download the Windows Installer version from http://www.hummingbird-alm.com, and launch it after download.

First choose the language during the installation

Then choose the folder where all Hummingbird App and its components will be installed.

Click Next to follow the installation guide until the installation finishes.

Click Finish to launch Hummingbird after installation.

Updating and Upgrading

When you want to update or upgrade Hummingbird App to latest version, you should download the latest version of Windows Installer and run it again. All your settings, histories and user profile will be conserved.
